## 2026-06-13 — v1.1.0 — первый публичный релиз
Опубликовано на GitHub (github.com/verdyshd/freepalp, MIT, CI ubuntu/windows ×
py3.11/3.12). Стратегические дифференциаторы и доводка к публикации:
- **Teacher→skill** — успешный ретрай дистиллируется в переиспользуемый
  `SKILL.md` (процедура + инструменты + урок), инжектится в промпт при похожей
  задаче; дедуп с bump uses (`core/skill_library.py`)
- **MCP-клиент** — stdio JSON-RPC, `connect_all` регистрирует `mcp__server__tool`
  в `ALL_TOOLS`; `config/mcp_servers.json` (opt-in) (`core/mcp_client.py`)
- **Токен-стриминг** ответа в WebUI (groq/openai_compat с реассемблингом
  tool_calls/ollama/gemini); on_token→SSE
- **DAG-декомпозиция + параллельные субагенты** (Architect, semaphore cap=2,
  фолбэк провайдеров, детектор обещанных файлов)
- **Deep Research** с реальными источниками (детерминированный триггер форсит
  живой поиск, не выдумывает ссылки)
- **Поиск по истории (FTS5)**, **артефакты-превью** (iframe-песочница),
  **MiMo-чекпойнты** длинных сессий, **бенчмарк-harness**
- **OpenAI-совместимый** `/v1/chat/completions` + `/v1/models` — любой IDE-плагин
  указывает base_url на оркестратор; **автозапуск Ollama**
- **WebUI**: 3 темы (светлая/тёмная/системная) + автоязык RU/EN/ZH по локали ПК
  с переключателем; UI добавления своих навыков и MCP кликом
- Безопасность: `session_keys` не отдаёт хост-ключи гостевым сессиям;
  README EN/RU/ZH; чистка истории (единый автор), gitignore артефактов сборки
