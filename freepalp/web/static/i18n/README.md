# FreePalp — переводы интерфейса (community i18n)

Интерфейс FreePalp можно перевести на **любой язык**, не трогая код — просто положив сюда JSON-файл.

Встроенные языки: **English (en), Русский (ru), 中文 (zh)**. Всё остальное добавляется файлами в этой папке.

## Как добавить свой язык (3 шага)

1. **Возьми шаблон.** Открой WebUI → ⚙️ Настройки → 🌐 Язык → **«📥 Экспорт шаблона»**.
   Скачается `freepalp-i18n-template.json` — все ключи интерфейса со значениями на английском.

2. **Переведи значения.** Переименуй файл в `<код>.json` (ISO-код языка, напр. `de.json`, `fr.json`, `es.json`, `it.json`)
   и переведи правые части. Можно перевести **не всё** — непереведённые ключи автоматически
   показываются по-английски. Положи файл в эту папку `freepalp/web/static/i18n/`.

3. **Зарегистрируй язык** в [`manifest.json`](manifest.json) — добавь объект в массив `languages`:
   ```json
   { "code": "de", "name": "Deutsch", "short": "DE" }
   ```
   - `code` — имя файла без `.json` и код для переключателя;
   - `name` — как язык подписан в Настройках;
   - `short` — короткая метка для меню-шестерёнки (необязательно).

Перезагрузи страницу — язык появится в переключателе (меню 🐙-шестерёнки и в Настройках).
Кнопка **«👁 Примерить файл»** в Настройках позволяет проверить перевод сразу, до сохранения на диск.

## Формат файла

Плоский JSON `{"ключ": "перевод"}`. Ключи — те же, что в шаблоне. HTML в значениях допустим
(некоторые строки содержат `<b>`, `<code>` и т.п. — сохраняй разметку, переводи только текст).
Эмодзи в начале строк лучше оставлять как есть.

```json
{
  "nav.chat": "Chat",
  "office.st.thinking": "Denkt nach…"
}
```

Пример живого частичного перевода — [`de.json`](de.json) (немецкий).

## Прислать перевод в проект

Сделай PR в [github.com/verdyshd/freepalp](https://github.com/verdyshd/freepalp) с твоим
`<код>.json` и строкой в `manifest.json` — добавим в официальную поставку. Спасибо! 🐙
