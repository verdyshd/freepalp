# FreePalp AI — Быстрый старт

## Требования
- **Windows 10/11** (или macOS/Linux)
- **Python 3.10+** → https://python.org/downloads/ (при установке отметь "Add to PATH")

---

## Запуск (Windows)

**Двойной клик на `start.bat`** — всё установится и запустится автоматически.

Браузер откроется сам на `http://localhost:8000`

---

## Добавление API ключей

Без ключей приложение не будет работать. Получи хотя бы один бесплатный:

| Провайдер | Лимит | Ссылка |
|-----------|-------|--------|
| **Groq** (рекомендую) | 14 400 req/day бесплатно | https://console.groq.com/keys |
| Cerebras | 30 000 req/day бесплатно | https://cloud.cerebras.ai/ |
| Google Gemini | 1 500 req/day бесплатно | https://aistudio.google.com/apikey |
| SambaNova | бесплатно | https://cloud.sambanova.ai/ |
| OpenRouter | бесплатные модели | https://openrouter.ai/keys |

После получения ключа:
1. Открой приложение → вкладка **Провайдеры**
2. Найди нужного провайдера → введи ключ → нажми **Добавить**
3. Нажми 🧪 чтобы проверить что ключ работает

---

## Запуск (macOS / Linux)

```bash
pip install -r requirements.txt
cp .env.example .env
python -m freepalp.gateway
```

Открыть: http://localhost:8000

---

## Если что-то не работает

- **"Python не найден"** → переустанови Python, обязательно с галочкой "Add to PATH"
- **Порт занят** → перезапусти `start.bat`, он освободит порт автоматически
- **Нет моделей** → добавь хотя бы один API ключ во вкладке Провайдеры
